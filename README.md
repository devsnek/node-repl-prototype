# Node.js REPL Prototype

Goals:

- Better debugging and interaction
  - Language Server Protocol
  - Runtime inspection
  - Benchmarking
- Pretty UI
  - Highlight output *and* input
  - autocomplete
- Keep the code neat for future changes

## Usage

### Install
```sh
$ npm install -g @nodejs/repl
```
```sh
$ node-prototype-repl
```
